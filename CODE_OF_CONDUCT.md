# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)
version 2.1.

In short: be respectful, assume good intent, keep discussion technical and
constructive, and welcome newcomers and institutions of every size. Harassment
or discrimination of any kind is not tolerated.

Report concerns to the maintainers at conduct@credentialcommons.org.

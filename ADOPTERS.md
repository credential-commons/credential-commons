## How adoption is measured (transparently)

Truly open code cannot — and should not — phone home. So adoption is measured by
signals that respect that:

- **This list** (self-registered, opt-in).
- **GitHub** stars / forks / "Used by" (dependency graph) / Action usage.
- **npm** download counts (once the package is published).
- **Published Linked Data in the wild** — because adopters reference the public
  `@context` URL, conformant data is discoverable by crawling for it. On-brand:
  adoption = who publishes interoperable data.
- **Opt-in** anonymous usage ping — only in the *hosted* service
  ([credentialstudy.com](https://credentialstudy.com)), never in this open CLI.

We deliberately do **not** embed tracking in the open-source tools. Trust is the
product.
